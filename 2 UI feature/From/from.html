User Input

🔹 Form

Definition
A Form is used to collect user input in a React application.
👉 Simple
A form is used to get information from the user.
For example, name, email, password, and phone number.
Why we use Form?
We use forms to collect and submit user data.
👉 Simple
Forms help users enter data and send it to the application.
It is commonly used in login, registration, and contact pages.

Common form elements are input, textarea, select, and button.

Form Handling
React uses event handlers like onChange and onSubmit to handle forms.
👉 Simple
onChange is used to track input changes.
onSubmit is used when the form is submitted.
Validation
Validation is used to check whether the entered data is correct.
👉 Simple
It helps prevent invalid or empty data.
For example, checking if an email format is correct.
Example Usage
Forms are used in login pages, registration pages, search bars, and contact forms.
👉 Simple
Most websites use forms to collect user information.
Real Meaning
A form is a way for users to interact with the application by entering data.
👉 Simple
Forms allow users to send information to the application.

Missing Concepts
🔹 Controlled Components
In React, form inputs are usually controlled by state.
👉 Simple
Input value React state-la store aagum.

Example:

const [name, setName] = useState("");

<input
  value={name}
  onChange={(e) => setName(e.target.value)}
/>
🔹 Uncontrolled Components
Sometimes form data is handled using refs instead of state.
