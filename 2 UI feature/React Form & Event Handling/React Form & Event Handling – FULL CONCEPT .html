React Form & Event Handling – FULL CONCEPT

1️⃣ FORM – Definition

👉 React form is used to collect user input and manage it using state and events.

👉 Simple Explanation

Tamil:

User input (text, email, password etc.) collect panna use panradhu form.

English:

Forms are used to take input from users and process it in React.

2️⃣ Why Forms in React?

👉 Real apps:

  • Login page
  • Signup page
  • Search box
  • Contact form

👉 All need forms

3️⃣ Controlled Component (VERY IMPORTANT 🔥)

👉 Input value is controlled by React state

Example:

import { useState } from "react"; function App() { const [name, setName] = useState(""); return ( <div> <input type="text" value={name} onChange={(e) => setName(e.target.value)} /> <h1>{name}</h1> </div> ); }

🧠 Meaning:
input value → state control
onChange → update state
UI always synced with state

4️⃣ Uncontrolled Component (less used)

👉 DOM directly handle input (useRef use pannuvanga)

import { useRef } from "react"; function App() { const inputRef = useRef(); const handleSubmit = () => { alert(inputRef.current.value); }; return ( <> <input ref={inputRef} /> <button onClick={handleSubmit}>Submit</button> </> ); }

🟢 Difference:

ControlledUncontrolled
useStateuseRef
React controls inputDOM controls input
RecommendedRare use

5️⃣ EVENT HANDLING – Definition

👉 Event handling means responding to user actions like click, input change, submit etc.

👉 Simple Explanation

Tamil:

User action (click, type, submit) nadandha React respond pannum.

English:

Event handling is used to handle user interactions in React.

6️⃣ Common Events in React

  • onClick → button click
  • onChange → input typing
  • onSubmit → form submit
  • onMouseOver → hover
  • onKeyDown → keyboard press

7️⃣ Basic Event Example

function App() { function handleClick() { alert("Button Clicked!"); } return ( <button onClick={handleClick}> Click Me </button> ); }

8️⃣ Event with Parameter

function App() { function greet(name) { alert("Hello " + name); } return ( <button onClick={() => greet("Rocky")}> Greet </button> ); }

9️⃣ Form Submit Handling

function App() { const handleSubmit = (e) => { e.preventDefault(); alert("Form Submitted"); }; return ( <form onSubmit={handleSubmit}> <input type="text" /> <button type="submit">Submit</button> </form> ); }

🧠 Important:

👉 e.preventDefault() = page reload stop pannum

🔟 Form + State (REAL USE)

import { useState } from "react"; function App() { const [email, setEmail] = useState(""); const handleSubmit = (e) => { e.preventDefault(); alert(email); }; return ( <form onSubmit={handleSubmit}> <input type="email" value={email} onChange={(e) => setEmail(e.target.value)} /> <button type="submit">Submit</button> </form> ); }

🚀 Interview Answer (PERFECT)

👉 Forms in React are used to collect user input and manage it using state or refs. Event handling is used to respond to user actions like clicks, input changes, and form submission. In React, we commonly use controlled components where form inputs are linked with state using onChange events. The onSubmit event is used to handle form submission, and preventDefault is used to stop page reload.
