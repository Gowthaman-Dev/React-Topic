useEffect FULL CONCEPT (React)

1️⃣ Definition

👉 useEffect is a React Hook used to handle side effects in functional components after rendering.

2️⃣ Simple Explanation

👉 Tamil:

Component load ஆன பிறகு அல்லது update ஆன பிறகு சில வேலைகள் (API call, timer, DOM update) செய்ய useEffect use pannuvanga.

👉 English:

useEffect is used to perform side effects like API calls, timers, and DOM updates after the component renders.

3️⃣ Why useEffect?

👉 React component render aana odane:

  • API call panna
  • Timer set panna
  • Event listener add panna
  • Data fetch panna

👉 idhellam side effects

4️⃣ Basic Syntax

useEffect(() => { // code here });

5️⃣ Types of useEffect (VERY IMPORTANT 🔥)

🟢 1. Run after EVERY render

useEffect(() => { console.log("Runs every time"); });

👉 No dependency array → every render

🔵 2. Run ONLY ONCE (Component mount)

useEffect(() => { console.log("Runs only once"); }, []);

👉 Empty dependency array = first time only

🟣 3. Run when DEPENDENCY changes

useEffect(() => { console.log("Count changed"); }, [count]);

👉 count change aana odane run aagum

6️⃣ Real Example (Counter)

import { useState, useEffect } from "react"; function Counter() { const [count, setCount] = useState(0); useEffect(() => { console.log("Count updated:", count); }, [count]); return ( <div> <h1>{count}</h1> <button onClick={() => setCount(count + 1)}> Increase </button> </div> ); }

7️⃣ Cleanup Function (IMPORTANT 🔥)

👉 Used to remove timers, listeners

useEffect(() => { const timer = setInterval(() => { console.log("Running..."); }, 1000); return () => { clearInterval(timer); }; }, []);

🧠 Meaning:

👉 Component remove aagumbodhu cleanup run aagum

3️⃣ Cleanup Function Runs When?

Most people think only on unmount.
Not fully correct.

useEffect(() => { console.log("Effect"); return () => { console.log("Cleanup"); }; }, [count]);

Flow:

Effect Runs ↓ count changes ↓ Cleanup Runs ↓ Effect Runs Again

Interview Answer
Cleanup runs:
✅ Before unmount
✅ Before the next effect execution

6️⃣ Missing Dependency Problem

Very Important 🔥

useEffect(() => { console.log(count); }, []);

But using:
count inside effect.

Problem
Effect may use stale values.

Fix

useEffect(() => { console.log(count); }, [count]);

Interview Point
Always include dependencies used inside the effect.

7️⃣ Infinite Loop Problem
