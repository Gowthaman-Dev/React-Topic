React Router DOM (Full Concept – Interview + Class Ready)

1️⃣ What is React Router DOM?

👉 React Router DOM is a library used in React to handle navigation (routing) between different pages without reloading the browser.

👉 Simple Meaning:

React app-la multiple pages maari behave panna use pannuvanga.

But actual-a:
👉 Single Page Application (SPA)

2️⃣ Why we use it?

👉 Without React Router:

Page change panna → full reload aagum ❌

👉 With React Router:

Page reload illa ❌
Fast navigation ✅
Smooth UI experience ✅

🧠 Simple English:

We use React Router DOM to navigate between different components (pages) in a React application without reloading the page.

7️⃣ Dynamic Routing (Important 🔥)

👉 URL parameter pass panna use
