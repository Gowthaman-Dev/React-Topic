3️⃣ Two ways to call API

🟢 1. Fetch API (Built-in browser method)

4️⃣ FETCH API

👉 Default JavaScript method

🔥 GET Request (Fetch)

import { useEffect, useState } from "react"; function App() { const [data, setData] = useState([]); useEffect(() => { fetch("https://jsonplaceholder.typicode.com/posts") .then((res) => res.json()) .then((data) => setData(data)); }, []); return ( <div> {data.map((item) => ( <h3 key={item.id}>{item.title}</h3> ))} </div> ); }

5️⃣ AXIOS (Most used 🔥)

👉 Easy + clean + industry standard

Install:

npm install axios

🔥 GET Request (Axios)

import axios from "axios"; import { useEffect, useState } from "react"; function App() { const [data, setData] = useState([]); useEffect(() => { axios.get("https://jsonplaceholder.typicode.com/posts") .then((res) => setData(res.data)); }, []); return ( <div> {data.map((item) => ( <h3 key={item.id}>{item.title}</h3> ))} </div> ); }

6️⃣ POST REQUEST (Data send)

👉 Backend-ku data send panna use

Axios POST

7️⃣ PUT REQUEST (Update data)

axios.put("https://jsonplaceholder.typicode.com/posts/1", { title: "Updated Title", });

8️⃣ DELETE REQUEST

axios.delete("https://jsonplaceholder.typicode.com/posts/1");

9️⃣ Async Await (VERY IMPORTANT 🔥)

👉 Modern way to handle API

GET with async/await

import axios from "axios"; import { useEffect, useState } from "react"; function App() { const [data, setData] = useState([]); const fetchData = async () => { const res = await axios.get("https://jsonplaceholder.typicode.com/posts"); setData(res.data); }; useEffect(() => { fetchData(); }, []); return ( <div> {data.map((item) => ( <h3 key={item.id}>{item.title}</h3> ))} </div> ); }

🔟 ERROR HANDLING (VERY IMPORTANT 🔥)

👉 API fail aana handle panna

const fetchData = async () => { try { const res = await axios.get("https://jsonplaceholder.typicode.com/posts"); setData(res.data); } catch (error) { console.log("Error:", error); } };

1️⃣1️⃣ LOADING STATE (REAL PROJECT MUST 🔥)

👉 Data varra varaiku loading show panna

const [loading, setLoading] = useState(true); const fetchData = async () => { try { const res = await axios.get("https://jsonplaceholder.typicode.com/posts"); setData(res.data); } catch (error) { console.log(error); } finally { setLoading(false); } };

UI:

return loading ? ( <h2>Loading...</h2> ) : ( data.map((item) => <h3 key={item.id}>{item.title}</h3> ));

2️⃣ Why useEffect for API Calls? 🔥

useEffect(() => { fetchData(); }, []);

Why?

👉 Component render aana odane API call panna.

Interview Answer

👉 We usually place API calls inside useEffect so they execute after the component mounts and avoid unnecessary API requests on every render.

Interview Questions Missing 🔥

Why use Axios instead of Fetch?

👉 Axios provides automatic JSON transformation, interceptors, better error handling, request cancellation, and cleaner syntax.

Why use async/await?

👉 It makes asynchronous code easier to read and maintain compared to promise chaining.

Why use loading state?

👉 To show feedback to the user while data is being fetched.

Why use try/catch?

👉 To handle API errors gracefully and prevent application crashes.
