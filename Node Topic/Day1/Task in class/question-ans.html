Interview Questions & Answers

🔵 FRONTEND (React / JS)

1️⃣ What are all the hooks you know?

Answer (English):
Hooks are functions that let us use state and other React features in functional components.

Tamil:
Hooks na functional component-ல state, lifecycle, routing மாதிரி features use பண்ண உதவும் functions.

Common Hooks:

  • 🔹 useState – state manage
  • 🔹 useEffect – lifecycle / side effects
  • 🔹 useContext – global data sharing
  • 🔹 useRef – DOM reference
  • 🔹 useCallback – function optimization
  • 🔹 useMemo – value optimization
  • 🔹 useNavigate – routing (React Router)

2️⃣ Use of Frontend

Answer (English):
Frontend is used to create the user interface and interact with users.

Tamil:
Frontend user பார்க்குற screen, buttons, forms, UI create பண்ண use ஆகும்.

Examples:

  • 🔹 Login page
  • 🔹 Dashboard
  • 🔹 Forms
  • 🔹 Buttons, animations

3️⃣ Difference between React and JavaScript

JavaScript React
Programming language JavaScript library
Used for logic Used for UI building
DOM manipulation manually Virtual DOM
Slow for large apps Fast & optimized

Tamil:
JavaScript logic எழுத use ஆகும்.
React UI easy-ஆ, fast-ஆ build பண்ண use ஆகும்.

4️⃣ What is CSR (Client Side Rendering)?

Answer (English):
CSR means rendering happens in the browser using JavaScript.

Tamil:
CSR-ல HTML browser-க்கு போன பிறகு JavaScript UI build பண்ணும்.

Example:
React SPA applications

5️⃣ How do you pass data to backend?

Answer (English):
Data is passed using HTTP requests.

Tamil:
Frontend-ல இருந்து backend-க்கு data API call மூலம் அனுப்புவோம்.

Methods:

  • 🔹 GET – get data
  • 🔹 POST – send data
  • 🔹 PUT – update data
  • 🔹 DELETE – delete data

Tools:
fetch, axios

🟢 BACKEND (Node.js)

1️⃣ What is Node.js?

Answer (English):
Node.js is a JavaScript runtime environment used to run JavaScript outside the browser.

Tamil:
Node.js browser இல்லாம JavaScript run பண்ண உதவும் runtime.

2️⃣ What are all the imported modules in Node?

Built-in Modules:

  • 🔹 http – server create
  • 🔹 fs – file system
  • 🔹 path – file paths
  • 🔹 os – system info
  • 🔹 events – event handling

External Modules:

  • 🔹 express
  • 🔹 mongoose
  • 🔹 cors
  • 🔹 dotenv

3️⃣ Difference between Node.js and JavaScript

JavaScript Node.js
Runs in browser Runs on server
Frontend logic Backend logic
No file access File system access

Tamil:
JavaScript browser-ல run ஆகும்.
Node.js server-ல run ஆகும்.

4️⃣ What is Server?

Answer (English):
A server is a system that receives requests and sends responses.

Tamil:
Server client request accept பண்ணி response send பண்ணும்.

Example:
Browser → Server → Response

5️⃣ Backend process in Node.js

Answer (English):
Backend process handles API, database, authentication, and business logic.

Tamil (Flow):

  • 1️⃣ Client request
  • 2️⃣ Server receive request
  • 3️⃣ Logic execute
  • 4️⃣ Database operation
  • 5️⃣ Response send to client

Flow:
Frontend → API → Node Server → Database → Response

🔥 ONE-LINE INTERVIEW SUMMARY

  • 🔹 Frontend → UI & user interaction
  • 🔹 React → UI library
  • 🔹 CSR → Rendering in browser
  • 🔹 Node.js → Backend runtime
  • 🔹 Server → Request & response handler
