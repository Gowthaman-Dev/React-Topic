📝 Node.js HTTP Server Notes

1️⃣ Node.js Modules

Node.js-ல் modules இரண்டு விதமாக இருக்கலாம்:

Type Syntax Explanation
CommonJS const http = require("http"); பழைய Node.js style. Companies இல் still common.
ES Modules (ES6+) import http from "http"; Modern JS. import/export பயன்படுத்தலாம்.

Export Types:

Named Export

export const myVar = 10;

Default Export

export default app;

Tamil:

  • 🔹 Named export → ஒரே file-ல் பல values export பண்ணலாம்.
  • 🔹 Default export → ஒரே value export பண்ணலாம்.

2️⃣ HTTP Module

Node.js built-in module.

Purpose: Server create பண்ண, request handle பண்ண.

const http = require("http"); // CommonJS
// OR
import http from "http"; // ES Module

3️⃣ Create HTTP Server

const app = http.createServer((req, res) => {
  res.writeHead(200, { "Content-Type": "text/plain" });
  res.end("Hello World!");
});

app.listen(5000, () => {
  console.log("Server is running on port 5000");
});

Step by Step Explanation

Code Explanation Tamil
http.createServer() Creates a server object ஒரு HTTP server object உருவாக்கும்
(req, res) => {} Callback function for requests Request & Response கையாளும் function
res.writeHead(200, {...}) Send status code & headers Response status & type define பண்ணும்
res.end("Hello World!") Send response message Clientக்கு message அனுப்பும்
app.listen(5000, ...) Server start listening on port 5000 5000 portல் server run ஆகும்

4️⃣ Console Debugging

console.log(app);
  • 🔹 Prints server object for debugging.
  • 🔹 Useful to know server properties.

5️⃣ File System (FS Module)

Node.js-ல் FS module files read/write பண்ண use ஆகும்.

Example:

import fs from "fs";

// Read file
fs.readFile("example.txt", "utf8", (err, data) => {
  if (err) throw err;
  console.log(data);
});

Tamil:

  • 🔹 FS module → file-ஐ manipulate பண்ண உதவும்.

6️⃣ Summary / Key Points

  • ✅ Node.js has built-in modules like http, fs.
  • ✅ Modules styles: CommonJS (require) vs ES Modules (import/export).
  • ✅ HTTP server: create → listen → respond.
  • ✅ Companies use CommonJS for stability, ES Modules for modern JS features.
