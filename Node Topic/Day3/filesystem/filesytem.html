📁 Node.js fs (File System) – Main Concepts

1️⃣ File Read (File-a read pannurathu)

Concepts:

fs.readFile()
fs.readFileSync()

Tamil:
File-la irukkura content-a Node use panni eduthukkurathu.

Example idea:
Text file open panni data print pannurathu.

2️⃣ File Write (File-la write pannurathu)

Concepts:

fs.writeFile()
fs.writeFileSync()

Tamil:
Pudhu file create pannurathu illa existing file-a overwrite pannurathu.

3️⃣ File Append (Extra content add pannurathu)

Concepts:

fs.appendFile()
fs.appendFileSync()

Tamil:
Already irukkura file-la content add pannurathu (delete pannama).

4️⃣ File Delete (File remove pannurathu)

Concepts:

fs.unlink()
fs.unlinkSync()

Tamil:
System-la irukkura file-a delete pannurathu.

5️⃣ Folder / Directory Concepts

Concepts:

fs.mkdir() – folder create
fs.rmdir() – folder delete
fs.readdir() – folder-kulla files list
fs.existsSync() – file/folder irukka-nu check

Tamil:
Folders create, delete, list pannurathu.

6️⃣ File Info / Stats

Concepts:

fs.stat()
fs.statSync()

Tamil:
File size, created time, file-aa illa folder-aa nu check pannurathu.

7️⃣ Rename / Move File

Concept:

fs.rename()

Tamil:
File name change pannurathu illa file-a vere location-ku move pannurathu.

8️⃣ Async vs Sync (🔥 VERY IMPORTANT)

Async Sync
Non-blocking Blocking
Callback / Promise Wait pannum
Server-ku best Learning-ku easy

Tamil:
Server apps-la Async fs methods use pannanum (performance nalla irukkum).

9️⃣ fs Promises (Modern way)

Concept:

import fs from 'fs/promises';

Methods:

fs.readFile()
fs.writeFile()
fs.appendFile()

Tamil:
Callback avoid panni async/await use pannurathu.

1️⃣0️⃣ Streams (Large files)

Concepts:

fs.createReadStream()
fs.createWriteStream()

Tamil:
Periya files (video, logs) memory overload illaama handle pannurathu.

🧠 Interview-ku One-Line Summary

Node.js fs module is used to read, write, update, delete files and manage directories in the server.
