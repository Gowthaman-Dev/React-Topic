4️⃣ fs.appendFile() – Async (MAIN PART) (15 mins)

Code (type pannitu explain pannunga):

import fs from 'fs';

fs.appendFile('log.txt', '\nUser logged in', (err) => {
  if (err) {
    console.log("Error:", err);
    return;
  }
  console.log("Data appended successfully");
});

console.log("Appending data...");

Line-by-line explanation:

  • 'log.txt' → target file
  • \n → new line-la add panna
  • callback → success / error

5️⃣ fs.appendFileSync() – Sync (10 mins)

Code:

import fs from 'fs';

fs.appendFileSync('log.txt', '\nUser logged out');
console.log("Append completed");

Explain:

  • File update mudinja apram thaan next line
  • Blocking method
