4️⃣ fs.readFile() – Async (MAIN PART) (15 mins)

Code explain pannitu type pannu 👇

import fs from 'fs';

fs.readFile('data.txt', 'utf8', (err, data) => {
  if (err) {
    console.log("Error:", err);
    return;
  }
  console.log(data);
});

console.log("File read started...");

Board explanation (line by line):

  • fs.readFile → file read pannum
  • 'utf8' → readable text format
  • err → error irundha
  • data → file content

🔥 Important demo moment:

👉 Output order-a show pannu:

“Paathinga? File read background-la nadakkuthu. Idhu thaan Async.”

5️⃣ fs.readFileSync() – Sync (10 mins)

Code:

import fs from 'fs';

const data = fs.readFileSync('data.txt', 'utf8');
console.log(data);

console.log("File read completed");

Explain:

  • readFileSync → wait pannum
  • File read mudinja apram thaan next line

Output:

Hello Students!
Welcome to Node.js class
File read completed

Tamil:

“Idhu blocking method.”
