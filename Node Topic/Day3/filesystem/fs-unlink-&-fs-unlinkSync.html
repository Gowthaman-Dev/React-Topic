4️⃣ fs.unlink() – Async (MAIN PART) (15 mins)

Code (type pannitu explain):

import fs from 'fs'; fs.unlink('temp.txt', (err) => { if (err) { console.log("Error:", err); return; } console.log("File deleted successfully"); }); console.log("Delete request sent...");

Line-by-line explanation:

  • fs.unlink → file delete pannum
  • Callback → success / error
  • Async → background-la work aagum
🔥 Demo moment:
👉 Run pannitu folder open panni kaami
👉 temp.txt illa-nu show pannunga

Tamil punch line:
“File instant-aa poiduchu. Undo illa.”

5️⃣ fs.unlinkSync() – Sync (10 mins)

import fs from 'fs'; fs.unlinkSync('sample.txt'); console.log("File deleted (Sync)");

Explain:

  • File delete mudinja apram thaan next line
  • Blocking method

6️⃣ Async vs Sync Comparison (5 mins)

fs.unlink fs.unlinkSync
Async Sync
Non-blocking Blocking
Callback No callback
Server-ku best Practice-ku easy

Tamil:
