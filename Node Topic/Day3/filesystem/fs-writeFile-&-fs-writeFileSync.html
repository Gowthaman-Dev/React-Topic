4️⃣ fs.writeFile() – Async (MAIN PART) (15 mins)

Code (type pannitu explain pannunga):

import fs from 'fs';

fs.writeFile('data.txt', 'Hello Node.js Students!', (err) => {
  if (err) {
    console.log("Error:", err);
    return;
  }
  console.log("File written successfully");
});

console.log("Writing file...");

Explain line by line (Board):

  • 'data.txt' → file name
  • 2nd param → content
  • callback → success / error

🔥 Demo moment:

  • File already irundhaa → content change aagum
  • File illaatti → auto create aagum

Output order:

Writing file...
File written successfully

Tamil punch line:

“Idhu Async – background-la file write nadakkuthu.”

5️⃣ fs.writeFileSync() – Sync (10 mins)

Code:

import fs from 'fs';

fs.writeFileSync('sync.txt', 'This is Sync Write');
console.log("File write completed");

Explain:

  • Write mudinja apram thaan next line execute aagum
  • Blocking method

Output:

File write completed

6️⃣ Overwrite Concept – MUST SHOW (5 mins)
