The path module helps to work with file and folder paths in Node.js.

✅ Solution:

path module

4️⃣ Import path module (2 mins)

import path from 'path';

Tamil:
path Node-oda built-in module — install panna vendam.

5️⃣ Core path Methods (MAIN PART) (25 mins)

1️⃣ path.basename() (5 mins)

path.basename('/user/files/data.txt');

Output:

data.txt

Tamil:
Full path-la irundhu file name mattum edukkum.

2️⃣ path.dirname() (5 mins)

path.dirname('/user/files/data.txt');

Output:

/user/files

Tamil:
File irukkura folder path edukkum.

3️⃣ path.extname() (5 mins)

path.extname('image.png');

Output:

.png

Tamil:
File extension edukkum.

Real use:

  • Image check
  • PDF / DOC validation

4️⃣ path.join() 🔥 MOST IMPORTANT (5 mins)

path.join('users', 'data', 'file.txt');

Output:

users/data/file.txt

Tamil punch line:

“Slash pathiya yosanai vendam — path.join pathiya paathukkum.”

5️⃣ path.resolve() (5 mins)

path.resolve('files', 'data.txt');

Output:

C:/project/files/data.txt

Tamil:
Absolute path create pannum.

6️⃣ __dirname & __filename (5 mins)

console.log(__dirname);
console.log(__filename);

Tamil explain:

  • __dirname → current folder
  • __filename → current file

🔥 Combine with path:

path.join(__dirname, 'data.txt');

7️⃣ path vs fs (Board clarity) (3 mins)

Module Use
fs File read/write/delete
path File address handle

Tamil:

“fs = work pannum, path = address sollum”
