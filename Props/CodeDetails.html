Tamil Explanation (Simple 🔥)

props oru object.

Andha object-kulla irukka values-a

direct-aa variables-aa eduthukradhu thaan destructuring.

1.First understand props enna?

Parent component:

<StudentList students={students} age={20} />

Child component-ku React send pannradhu 👇

props = {

  students: [...],

  age: 20

}

👉 props = oru object

2.Without destructuring (normal way)

const StudentList = (props) => {

  console.log(props.students);

  console.log(props.age);

  return <div />;

};

👉 Every time props. nu type panna vendiyadhu

3.With destructuring (clean way)

const StudentList = (props) => {

  const { students, age } = props;

  console.log(students);

  console.log(age);

  return <div />;

};

4.Direct-aa function argument-la destructuring

const StudentList = ({ students, age }) => {

  return (

    <div>

    {students.map(s => (

      <p>{s.name}</p>

    ))}

    </div>

  );

};

BEST PRACTICE (Interview Answer)

const StudentList = ({ students = [] }) => {

  return students.map(...);

};

default props la array of object ku errror varama iruka { students = [] } ippudi podanum naa sonnathu crt ta

1.map() naa enna?

👉 map() = array loop method

English:

map() is used to loop through an array and return a new array.

Tamil:

map() na array-oda ella values-um one by one loop panni,

pudhu result array create pannrathu.
