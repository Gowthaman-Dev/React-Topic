useRef UI Update Explanation

Method : 02

React Code

import { useEffect, useRef, useState } from "react"

function App() {

  const [count, setCount] = useState(0)
  const prevCount = useRef(0)

  useEffect(() => {
    prevCount.current = count
  }, [count])

  const handleClick = () => {
    setCount(prev => prev + 1)
  }

  return (
    <>
      <h1>Current Value : {count}</h1>
      <h2>Previous Value : {prevCount.current}</h2>

      <button onClick={handleClick}>Increment</button>
    
  )
}

🔥 useRef UI-la show aagaathu.

Appo kelvi:

👉 <h2>{prevCount.current}</h2> eppadi update aaguthu?

🎯 Answer

UI update aagurathu useState nala thaan.

useRef just value hold pannuthu.

Step by step flow (slow ah explain pannunga)

Initial:

count = 0

prevCount = 0

UI:

Current: 0

Previous: 0

Button click →

setCount() run aagum

🔥 Component rerender aagum (because useState)

Now:

count = 1

prevCount = 0

So UI:

Current: 1

Previous: 0

After render → useEffect run aagum:

prevCount.current = count

So:

prevCount = 1

⚠️ rerender illa (because ref)

So UI still:

Previous: 0

Next click:

Again setCount → rerender

Now React read pannum:

prevCount.current = 1

So UI shows:

Previous: 1

🔥 Main teaching sentence (MUST SAY)

useRef value change panninaa UI update aagaathu — but next state rerender la athu screen-la theriyum

👉 UI marumadi varathu

👉 ref value screen-la varum
